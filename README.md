# Code du cours de l'option VR pour M48

Create a folder for your project and :

### Clone, fork or download
**clone**:
```sh
git clone https://github.com/Chabloz/a-frame-48-cours.git .
```
### Install
```sh
npm install
```
### Dev
```sh
npm run dev
```
### Build
```sh
npm run build
```
